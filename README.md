## Структура источников

1252 документа в `data/raw/` охватывают:

- **Ежедневные рефлексии** (знания/feedback loop) — апрель–июнь 2026
- **Agent-framework исследования** — AutoGen, CrewAI, LangGraph, Agno, AutoGen, Magentic-One
- **S3C 排产** — проект производственного планирования (scheduling)
- **OpenClaw / ClawLink** — архитектура AI-агентов
- **Insight-документы** — синтезированные AI-аналитикой выводы
- **Технические заметки** — RAG, векторные БД, трансформеры, MCP

## Концепты (500)

Семантическая карта знаний. Каждый концепт содержит:
- `slug` — уникальный идентификатор
- `kinds` — тип (tag, section, keyword, phrase, source-title)
- `evidence` — цитаты из исходников с привязкой к источнику
- `signalScore` — метрика значимости
- `contradictions` — противоречия между источниками
- `mainAxis` — привязка к главной оси знаний
